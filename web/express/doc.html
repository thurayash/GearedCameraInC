Issues

Issues can occur, mainly due to a problem of configuration within your system. Colliding systems tend to birth problems like the following.

Software

Software issues [..]

Error code Cause Solution
1 File Not Opening / Found Re-download our files, and check the path (the location of the files). Additionally, try to compare to see if you have any missing files directly from the Github.
2 SDL Error Check if you have downloaded the right version of SDL, then check that you have set your SDL libraries correctly.
3 Image Not Opening / Found Check if the image you want to use is present, as well as the path.
4 Video Manipulation Error Check if all persmissionshave been given to GCC. Additionally, check that the camera is correctly configured and plugged, and that your memory storage devices are working. Disable Firewalls as a final solution.
5 Camera Not Found Unplug and replug the camera. Additonally, switch ports after cleaning both the male and female USB ports.

Hardware

Hardware issues [..]

Error code Cause Solution
3001 ERROR reading bytes. Unplug then plug the device. Make sure to launch th software after the first beeps.
3002 Could not compile regular expression. Check for the permission given to the software, push on the reset button and retry !
3003 GCC hardware need to be restarted Unplug the device, wait for 10 seconds. Plug it to the computer and keep pressing on the reset button for 5 seconds
3004 File descriptor corrupted Unplug the device and retry, if it still persist check for the permission, restart the computer and retry later.
3005/3006 Command access corrupted ! Make sure that all the dependencies are correctly installed by launching the config file.
3007 Connection failed ! The hardware has been disconnected from the computer, please reconnect it and retry later. If it's still disconnected please contact us.
3008 An error occured while setting /dev/ Make sure that the permission are given using chmod +xu ./main. If it's still not working, check if the compuer is compatible.
